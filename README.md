#### 트레이닝 기능 사용법
- 트레이닝 기능만 사용
  - BoB-Linux/Workspace/BoB Trainer 디렉토리에서 make all
  - 그 후 생성된 trainer 파일을 실행하면 됨
  
- BoB Linux에서 사용
  - 콘솔 창에 'trainer'를 입력하면 트레이닝 기능 실행
  - \"/usr/local/bin/trainer\"에 위치
