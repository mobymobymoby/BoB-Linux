#명령어 정리
1. ifconfig: 네트워크 인터페이스 설정
- if = interface
- inet = internet
2. env: 환경 변수 확인
- $[변수이름]으로 접근 할 수 있다.
- [변수이름]=[값] 이퀄 기호에 띄어쓰기를 하지 않는다.
3. 파이프라인: '\'키에 있는 작대기 '|'
4. echo $[변수이름]: 변수이름에 해당하는 환경변수를 볼 수 있다.
5. vi 에디터
5.1. Normal mode: 처음 상태
- hjkl로 왼쪽, 아래, 위, 오른쪽 커서 이동
- yy: 한줄 복사 (yank)
- dd: 한줄 삭제
- p: 붙여넣기
- u: undo 되돌리기
5.2. Visual mode:
- i: insert
- a: append
- o: 다음 줄
5.3. Command mode:
- ':' 콜론을 입력하면 명령모드
- :[숫자] 해당 라인으로 이동
- :set nu 라인 넘버 보이기
- :$ 마지막 라인으로 이동
- /[찾을문자열] 이 후 'n'누르면 다음
- :S/[찾아]/[바꾸기]
- :1,$ S/[찾아]/[바꾸기]/g  1번 라인에서 끝 라인까지 바로 바꿈(go)
6. 리다이렉션
- 예) man ls > abc.txt
- 화면(stdout)으로 출력되는 것을 파일로 갔다. 그것이 리다이렉션
- '>': stdout만 파일로 간다.
- '>&': stdout, stderr도 파일로 간다.
- '>>': 덮어쓰지 않고 추가
7. 리소스파일
- '.'으로 시작하고 'rc'로 끝나는 '.~rc'파일
8. ls
- -F: 디렉토리는 뒤에 '/'를 붙임
- -R: 디렉토리 안에 까지 전부 (recursion?)
- -r: 내림차순으로 보여줌 (reverse)
- -i: inode 값 보여줌
- -al: 자주 쓰는 조합
9. cp
- -p: 시간까지 똑같이 복사 (preserve)
- -l: 하드링킹 inode 값도 같게 복사
- -s: 소프트링크, 심볼릭링크
10. ln: 하드링킹
- -s: 소프트링크
11. cat
- -n: 라인 넘버 표시, 공백 줄도 표시
- -b: 라인 넘버 표시, 공백 줄은 무시
- -T: 탭 문자 표시
12. stat: 파일 상태 정보 확인
13. head: 파일 내용 앞부분
14. more: 파일 내용을 화면단위로
15. less: 파일 내용을 페이지 단위로
16. tail: 파일 내용 뒷부분
- f: 내용 바뀌는걸 계속 볼 수 있음
17.
- redirection: 파일에 연결
- pipeline: 프로세스에 연결
18. grep: 해당 문자열 패턴이 나타나는 라인만 보여줌
19. diff: 줄 단위로 파일 내용을 비교
20. ps: 프로세스 목록
- -e: 커널 프로세스를 제외한 모든 프로세스
- -f: 전체 명령어 상태
- -l: 상세정보
- -aux: 자주 쓰는 조합
21. tar
- -c: 압축?
- -x: 해제
- -f: 파일명 지정
- -v: 화면에 출력
- -z: zip파일
22. git
- add -> commit -> push
23. make
- 컴파일을 쉽게
- makefile을 배워야 함
===============================
24. which: 명령어의 경로를 확인
25. alias: 자주 사용하는 명령어를 특정 문자로 입력해 두고 대신 사용
26. unalias: alias 기능을 해제
27. 환경변수 PATH: 명령어 디렉토리들
28. man: 명령어들의 매뉴얼을 제공
29. info: man에 비해 한정적
30. whatis: 명령어에 대한 기능을 간략하게 나타냄
31. manpath: man 페이지의 경로
32. whereis: 명령어의 실행파일 경로, 설정 파일 및 매뉴얼 페이지를 검색
33. useradd: 계정 생성, 계정만 만듬
34. adduser: 차근차근 계정 생성
35. su(switch user): 현재 계정에서 로그아웃 하지 않고 다른 계정으로 로그인
36. 파일 /etc/default/useradd: useradd로 계정을 추가할 때의 기본값들
37. 파일 /etc/passwd: 계정 정보를 가지고 있다.
38. 파일 /etc/shadow: 패스워드 정보를 가지고 있다.
39. 파일 /etc/login.defs: 계정 설정 기본값
40. usermod: 사용자 정보 변경
41. userdel: 계정 삭제
42. change: 패스워드 만료 정보를 변경
43. 파일 /etc/group: 그룹들이 정의 되어 있다.
44. groupadd: 그룹 추가
45. groupdel: 그룹 삭제
46. groupmod: 그룹 정보 변경
47. users: 시스템에 로그인한 사용자 정보 출력
48. who: 현재 접속해 있는 사용자 조회
49. w: 명령어는 짧지만 더 자세하게 조회
50. id: UID, GID, group 확인
51. group: 속한 그룹 목록 확인
52. ping: 때리기
53. traceroute: 목적지까지 경로를 표시
54. nslookup: 도메인->ip 혹은 반대
55. dig, host도 비슷
56. hostname: 시스템 이름을 확인 변경
57. shutdown: 종료, 재부팅
58. init: shutdown과 유사
59. reboot: 재부팅
60. halt: 종료
61. chown: 소유자 변경
62. chmod: 퍼미션 변경
63. umask: 새로 생성되는 파일이나 디렉터리의 기본 퍼미션 설정
- 디렉토리는 777, 파일은 666에서 umask를 뺀 값
64. SetUID, SetGID, sticky bit 내용 솔직히 모르겠어요
65. mount: 파일시스템을 마운트
65. umount: 언마운트지만 명령어는 유마운트
66. fdisk: 파티션 설정
67. mkfs: 파일 시스템 생성
68. fsck: 파일 시스템 점검 및 복구
69. df: 남은 용량 확인
- -h: KB, MB, GB로 표현
70. 파일 /etc/fstab: 파일 시스템 정보를 고정적으로 저장
71. 쉘 관련
- /etc/shells 사용가능한 쉘 목록
- /etc/passwd 계정별 기본 쉘 확인 가능
- echo $SHELL 내가 무슨 쉘인지 확인
72. !!: 마지막 명령문 실행
73. !n: n번째 명령문 실행
74. !string: 'string'으로 시작하는 최근 명령문 실행
75. ps: 현재 실행 중인 프로세스의 상태를 보여줌
76. pstree: 프로세스를 트리구조로 보여줌
77. jobs: 중지, 백그라운드인 프로세스를 보여줌
78. bg, fg: 작업 전환
79. kill: 시그널 보내 시그널 보내 찌릿 찌릿 찌릿 찌릿
80. nice: 프로세스의 우선순위 변경
81. renice: 마찬가지 우선순위 변경, nice와는 값이 다름
82. top: 시스템 운영상태를 모니터링
83. nohup: 로그아웃하거나 터미널 닫혀도 백그라운드로 작업
84. 파일 /etc/crontab: 주기적으로 수행할 작업 설정, crond에 의해 수행
