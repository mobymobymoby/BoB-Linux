이번에 학습할 명령어는 'sort' 입니다. sort는 파일의 내용을 행 단위로 끊어서 정렬을 해줍니다. 정렬된 결과를 출력만 해줄 뿐, 따로 저장하거나 원본파일을 변경하지는 않습니다.

sort 명령어는 다음과 같은 형식으로 사용합니다.
$ sort [옵션] 파일

우선 cat으로 정렬 전의 'sort_test_file' 파일의 내용을 확인해봅시다.@

이번엔 sort를 이용하여 'sort_test_file' 파일을 내용을 정렬시켜 확인해보세요.@

sort 명령어의 옵션은 아래와 같습니다.
-b : 공백은 무시
-d : 사전 순으로 정렬
-f : 대소문자 무시
-n : 숫자를 기준으로 정렬
-r : 내림차순으로 정렬

sort 명령어는 정렬하여 출력만 해주므로, 파이프라인|  혹은 리다이렉션>  기법과 같이 많이 사용됩니다.
