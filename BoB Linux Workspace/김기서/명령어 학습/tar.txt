이번에 학습할 명령어는 'tar'입니다.
tar 명령어는 Tape ARchive의 약자로, 여러 파일들을 하나의 파일로 묶는 명령어 입니다.

tar 명령어의 옵션은 아래와 같습니다.
-c, --create : 새로운 tar 파일을 생성
-f, --file : tar 파일의 이름을 지정
-j, --bzip : bzip2를 이용한 압축
-t, --list : tar 파일 내용을 출력
-v, --verbose : 상세한 정보를 출력
-x, --extract : tar 파일을 푼다.
-z, --gzip : gzip을 이용한 압축

tar 명령어는 단독으로 사용되거나 하나의 옵션만으로 사용되는 경우는 거의 없습니다.
다양한 옵션의 조합으로 사용됩니다.
대표적인 조합 두 가지는 아래와 같습니다.
-cvf : 여러 파일을 하나의 tar 파일로 묶을 때 사용
-xvf : 묶여있는 tar 파일을 풀 때 사용
이 두 옵션은 잘 기억☆해주셨으면 합니다.

tar 명령어는 다음과 같이 사용합니다.
tar [옵션] 결과파일이름 묶을파일1 묶을파일2 ...

tar -cvf 를 이용하여 file1, file2, file3를 'test.tar'로 묶어보세요.@

이번에는 tar -lf 를 이용하여 'test.tar'의 내용을 확인해보세요.@

방금 전에 하나로 합친 파일 들을 볼 수 있습니다.
그러면 다시 tar -xvf 를 이용하여 'test.tar'를 풀어봅시다.@

단순히 tar 로 묶는 것으로는 압축은 되지 않습니다. tar는 어디까지나 묶어만 줍니다.
압축은 -j 혹은 -z 옵션을 덧붙여서 이뤄집니다.

tar 명령어에 대한 학습이 끝났습니다.
