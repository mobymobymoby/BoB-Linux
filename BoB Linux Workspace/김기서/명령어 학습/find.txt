이번에 학습할 명령어는 'find'입니다. 이름에서 알 수 있듯이 파일을 찾는 명령어입니다.

find 명령어는 다음과 같은 형식으로 사용합니다.
$ find [찾을 경로] [옵션]

[찾을 경로]를 생략하면 현재 디렉토리에서 부터 검색을 합니다.

find 명령어의 옵션은 아래와 같습니다.
-name 찾을이름 : 이름으로 파일 찾기
-user 유저이름 : 소유자가 유저이름인 파일 찾기
-type [bcdfls] : 파일 타입으로 파일 찾기
    b: 블록파일
    c: 문자
    d: 디렉토리
    f: 파일
    l: 링크파일
    s: 소켓
-size [+/-]n : 파일 크기로 찾기
    +n : n보다 큰 파일
    -n : n보다 작은 파일
     n : n인 파일
