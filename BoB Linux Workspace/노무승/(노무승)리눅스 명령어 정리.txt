[기본 작업 명령어] (25개)
	cp		파일 복사
	mv		파일 이동
	ls -al / dir	파일 확인
	ln		심볼릭 링크 생성
	cd		작업 디렉터리 이동
	pwd		작업 디렉터리 출력
	mkdir		폴더 생성
	rmdir		폴더 삭제
	touch		파일 생성
	rm -rf		파일 영구 삭제
	sudo -s		루트 권한 상승 (쉘 전체 적용)
	sudo [명령어]	루트 권한 상승 (일회성)
	init [넘버]	시스템 종료
	shutdown	시스템 종료
	cat		파일 내용 출력
	find / -name "" 	파일 검색
	grep		특정 단어 검색
	./[ELF 파일]	파일 실행
	nano / vi / vim	텍스트 편집기
	man		설명서 출력
	[명령어] --version 	유틸 버전 확인
	chmod		파일의 권한 변경
	chown		파일 소유 권한 변경

[사용자 계정 관련 명령어] (9개)
	groupadd	사용자 그룹 추가
	groupdel		사용자 그룹 삭제
	groups		사용자 그룹 확인
	gpasswd		사용자 그룹 관리
	useradd		사용자 계정 추가
	userdel		사용자 계정 삭제
	passwd		사용자 패스워드 변경
	cat /etc/passwd	전체 사용자 목록 확인
	grep /bin/bash /etc/passwd useradd를 통해 등록된 사용자 목록 확인

[네트워크 명령어] (5개)
	ifconfig		네트워크 인터페이스 조회
	ping		icmp 패킷 전송
	wget		HTTP로 파일 다운로드
	netstat		프로세스 별 네트워크 연결 상태 출력
	nslookup		DNS 서버의 정보를 쿼리하기 위해 사용

[패키지 관련 명령어] (3개)
	apt-get update	패키지 리스트 업데이트
	apt-get upgrade	패키지 업그레이드
	apt-get install	리스트에 등록된 패키지 설치
	./configure	make 파일을 설정함.
	make (-j4)	설치 파일을 빌드함
	make install	빌드한 파일을 설치함.

[기본 프로그래밍 컴파일 관련 명령어] (3개)
	gcc / g++	C(++)언어 컴파일
	python		파이썬 컴파일

[압축 관련 명령어] (4개)
	tar -cfvz [파일명] [경로]	tar 압축
	tar -xvf [파일명]		tar 압축 해제
	gzip [파일명]		gz 압축
	gzip -d [파일명]		gz 압축 해제
	zip [파일명] [경로]		zip 압축
	unzip [파일명]		zip 압축 해제

[프로세스 관련 명령어] (3개)
	ps	프로세스 리스트와 상태 출력
	top	CPU 점유율, 메모리 사용률, 프로세스 리스트 출력
	kill [PID]	프로세스 종료
