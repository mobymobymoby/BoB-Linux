BoB Linux에 오신 것을 환영합니다.
본 파일은 Linux를 처음 사용해보는 입문자를 위한 파일입니다.
BoB Linux는 KITRI BoB 9th 프로젝트 팀 "Black Moon"이 만든 교육용 리눅스입니다.
보안을 공부하기 위해 리눅스를 사용하는 입문자들을 위해 트레이닝 기능을 제공하고 있습니다.

/* 트레이너 시작 방법 추가 */

트레이너를 시작하기에 앞서서, BoB Linux의 기본적인 사용 방법을 알려드리겠습니다.

1. 먼저. 본 리눅스는 Ubuntu 기반으로 제작되었습니다.
따라서 패키지 관리자는 우분투의 'apt'를 이용합니다.

2. 옵션의 중첩
트레이닝 기능에서 배우게 되는 대부분의 옵션은 중첩하여 사용할 수 있습니다.
예를 들어 'ls' 명령어의 '-a' 옵션과 '-l' 옵션은 "ls -a"이나 "ls -l"로 따로 쓸 수도 있지만 "ls -al"과 같이 한 번에 사용할 수 있습니다.
이 때의 기능은 -a 옵션과 -l의 옵션을 합친 것이 됩니다. 이러한 법칙은 ls 뿐만 아니라 대부분의 명령어에 통용됩니다.

3. 터미널에서의 스크롤 이동 방법
BoB Linux Trainer는 콘솔이나 터미널에서 진행됩니다.
키보드 방향키 위에 있는 "PgUp" 키와 "PgDn" 키를 이용해 스크롤을 이동할 수 있습니다.
<Shift + PgUp>은 스크롤을 위로, <Shift + PgDn>은 스크롤을 아래로 움직입니다.
"PgUp" 키와 \"PgDn\" 키가 없다면 "fn" 키와 키보드의 위, 아래 방향키를 조합하여 사용하시면 됩니다.\n");

4. 트레이닝 기본 디렉토리
트레이닝을 진행할 때는 사용자가 리눅스의 다른 디렉토리나 파일을 변형할 위험이 있어서 별도의 디렉토리로 지정된 상태에서 진행됩니다.
'home/[사용자 계정 이름]/tr' 디렉토리가 기본 디렉토리가 되어 명령어를 수행합니다.
이는 지정된 사용자 계정 이름과 함께 트레이닝 기능 중간에 명령어를 실습할 때 콘솔이나 터미널 화면에 출력됩니다.

5. [사용자 계정 이름]@localhost:/home/[사용자 계정 이름]/tr$
         (1)            (2)               (3)           (4)
트레이닝 진행 시 여러분의 입력이 필요한 부분에서는 위와 같은 메시지가 출력됩니다.
해당 입력 창을 '프롬프트'라고 부르며, 위에 보이시는 것과 같은 구조로 되어 있습니다.
프롬프트 창의 구조는 리눅스마다 다르지만, 거의 비슷한 구조를 따릅니다.

위에서는 크게 4가지로 분류할 수 있습니다.
(1) : 사용자의 계정 이름이 들어갑니다.
(2) : '호스트'의 이름이 들어갑니다.
(3) : 현재 작업 디렉토리가 표시됩니다.
(4) : 현재 사용자가 루트 계정이라면 #으로, 아니라면 $으로 표시됩니다.

6. '-' 옵션과 '--' 옵션(롱 옵션)
리눅스 명령어에는 다양한 기능을 처리하기 위한 옵션이 존재합니다.
보통 옵션은 '-a'와 같이 -(dash)를 이용하여 사용합니다.
하지만, '-'를 두 번 붙여서 사용하는 방법으로 옵션을 사용할 수도 있습니다.
예를 들어 ls 명령어의 '-a' 옵션을 롱옵션으로 사용하면 'ls --all'으로 입력하면 됩니다.
위의 두 표기법은 동일한 동작을 합니다.
대부분의 사람들은 더 짧은 옵션을 사용하지만 간혹가다가 롱 옵션을 사용하는 경우도 존재합니다.
